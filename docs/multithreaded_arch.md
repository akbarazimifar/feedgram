# Multithreaded Architecture

This is a detailed view of the architecture of the application.

It mainly consists in:

|  #  | Description                                                     |
|:---:|-----------------------------------------------------------------|
|  6  | **persistent threads**                                          |
| _n_ | **ephemeral threads** generated by the  _News Retreiver_ thread |
|  2  | **shared _queues_**                                             |
|  1  | **shared _list_**                                               |
